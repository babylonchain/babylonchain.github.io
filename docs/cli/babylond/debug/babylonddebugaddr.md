# Babylond_debug_addr
Convert an address between hex and bech32.
## debug addr command
```
babylond debug addr [address] [flag]
```
### Example Command
```
$ babylond debug addr cosmos12345tdzjh8p2xq95kk0exp7shwjklj
```
## Options
```
 -h, --help   help for addr
```
